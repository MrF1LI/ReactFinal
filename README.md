# ­ЪЇй№ИЈ FoodApp Рђћ React Meal Browser

рЃћрЃА рЃљрЃарЃўрЃА рЃцрЃўрЃюрЃљрЃџрЃБрЃарЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ React-рЃерЃў Рђћ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљ рЃџрЃљрЃЏрЃљрЃќрЃљрЃЊ рЃљрЃгрЃДрЃЮрЃЉрЃўрЃџрЃў Single Page Application (SPA), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА [TheMealDB API](https://www.themealdb.com/api.php) рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃарЃћрЃфрЃћрЃърЃбрЃћрЃЉрЃўрЃА рЃАрЃљрЃФрЃўрЃћрЃЉрЃџрЃљрЃЊ рЃЊрЃљ рЃЊрЃљрЃАрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЉрЃџрЃљрЃЊ.

---

## ­ЪћД рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

| рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ | рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ |
|------------|-------------|
| **React** (Vite) | рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў SPA |
| **React Router v6** | рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ |
| **React Context** | рЃЌрЃћрЃЏрЃўрЃА рЃЊрЃљ рЃћрЃюрЃўрЃА рЃњрЃџрЃЮрЃЉрЃљрЃџрЃБрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ |
| **Axios** | API-рЃЊрЃљрЃю рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ |
| **Framer Motion** | рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃБрЃарЃў рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃћрЃЉрЃў |
| **Bootstrap 5.3** | рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃЋрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў |
| **Custom Hooks** | рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃгрЃљрЃдрЃћрЃЉрЃљ `useFetch()`-рЃўрЃЌ |
| **LocalStorage** | рЃЌрЃћрЃЏрЃўрЃАрЃљ рЃЊрЃљ рЃћрЃюрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ |
